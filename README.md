# Clusterizador Housing Data

Este projeto tem como objetivo aplicar técnicas de **clusterização** em dados de habitação, permitindo identificar padrões e agrupamentos relevantes para análise exploratória.

## 🚀 Tecnologias Utilizadas
- Python 3.13
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- KMeans
- Math
- Pickle
- cdist
- MinMaxScaler
